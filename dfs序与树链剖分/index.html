dfs序与树链剖分

发表于 2020-07-21 更新于 2022-06-27

核心知识点

树链剖分主要可以解决一类树上的值问题，相当于把树上问题转换成了链上的问题。

dfs序

如果操作只涉及子树，那么我们只需要使用点在dfs序的位置内就可以完成操作了。

概念

dfs序，即遍历一个树的顺序。
那么，先根还是后根呢？答案是，先根（想一想，为啥呢

性质

每个点的子树，对应 DFS 序上一段连续的区间。

进入点 x 的时间为 dfnx，离开 x 的子树的时间为 dfnx+sizex−1\rm dfn_x + size_x -1dfnx​+sizex​−1

点 x 的子树在 DFS 序上对应的区间为 [dfnx,dfnx+sizex−1]\rm [dfn_x, dfn_x + size_x − 1][dfnx​,dfnx​+sizex​−1].

按照 DFS 序维护点的相关信息，子树操作变为序列区间操作。(win!)

--Powered by dalao cdsfcesf

具体实现

对一棵树进行 DFS。每访问到一个新的节点，记录下它的编号。

可以得到一个长为 n 的排列，称为这棵树的一个 DFS 序。

同时得到点 x 在 DFS 序中的位置编号dfnxdfn_xdfnx​

  2. LCA

    LCA就是最近公共祖先的意思。

    若 yyy 在 xxx 到根的路径上，且 x≠yx \neq yx​=y ,那么 yyy 是 xxx 的祖先

    xxx 的所有祖先是 xxx 到根的路径上除 xxx 之外的所有点。

    求点 zzz 满足 zzz 是 xxx 和 yyy 共同的祖先，且深度最大。则称 zzz 为 xxx 和 yyy 的最近公共祖先 (LCA)。

树链剖分具体实现

  1. 函数部分
    需要使用两遍dfs,
    首先 DFS 一次，求出每个点的子树大小和重儿子。
    再进行一次 DFS 记录 DFS 序。到达一个点时，先访问重儿子，
    再访问其余儿子。
    对一条重链，链上的点在 DFS 序中一定相邻。
    因此一条重链对应 DFS 序中一段区间。
    可用数据结构按照 DFS 序维护相关信息。
  2. 变量部分
    DFS 两次，求出重要的量：
    sonx\rm son_xsonx​：xxx 的重儿子
    dfnx\rm dfn_xdfnx​：xxx 在 DFS 序中的编号
    topx\rm top_xtopx​：xxx 所在重链的链顶
    若 (x,y)(x, y)(x,y) 是重边，则 topytop_ytopy​ = topxtopxtopx；
    若 (x,y)(x, y)(x,y) 是轻边，则 topytop_ytopy​ = yyy。
